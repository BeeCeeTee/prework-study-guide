Prework Study Guide

HTML

  • HTML stands for HyperText Markup Language, and represents the main structure of a website.
  • Every html document begins with a “!DOCTYPE html” declaration to tell the browser what type of file to expect.
  • The “head” element contains metadata and other information about the website.
  • The “body” element contains the portion of the site which is visible to the user.
  • The “body” element can further be divided into subsections called “child elements”
  • On this page, the “body” section is divided into three child elements: the “header”, “main” and “footer” elements

CSS

  • CSS stands for Cascading Style Sheets, and represents the visual design aspect of a website.
  • CSS can be implemented in three ways: inline CSS, internal CSS and external CSS.
  • CSS is most often implemented externally via a “style sheet”.
  • CSS can be used to define many style aspects of a website, including the margin (space around the outside of a site), padding (space around the inner edge of an element), the size and alignment of elements, colors, shading effects and more.
  • Effects can be applied to all elements using a wildcard selector (*) or by defining a specific class (.CLASSNAME) to apply effects only to specific elements.
